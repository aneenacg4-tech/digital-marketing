# 🚀 Landing Page Project

This is a responsive landing page built using **HTML** and **Tailwind CSS**. It includes a navigation bar, hero section, and basic styling with modern fonts and icons.

## 📌 Features

* Responsive navigation bar
* Hero section with heading, text, and call-to-action button
* Mobile-friendly design using Tailwind CSS
* Google Fonts integration (Poppins & Roboto)
* Font Awesome icons support

[live@]( https://aneenacg4-tech.github.io/digital-marketing/)

![ss3](./sv.png)

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS (via CDN)
* Google Fonts
* Font Awesome

## 📂 Project Structure

```
project-folder/
│── index.html
│── csss.jpg   (image used in hero section)
│── README.md
```

## ⚙️ Setup Instructions

1. Download or clone this repository.
2. Make sure all files are in the same folder.
3. Open `index.html` in your browser.

No build tools or installation required since Tailwind is used via CDN.

## 📱 Responsiveness

* Navigation menu adapts for mobile (hamburger icon shown)
* Layout switches between column and row for different screen sizes

## 🖼️ Customization

* Replace `Logo` text with your brand name
* Update the image (`csss.jpg`) with your own asset
* Modify colors and fonts via Tailwind config in the `<script>` section
